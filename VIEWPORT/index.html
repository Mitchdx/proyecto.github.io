<Viewport>

El viewport es el área visible de una página web dentro de una ventana del navegador. Es, básicamente, la parte de la página que el usuario puede ver sin desplazarse. El tamaño del viewport puede variar dependiendo del dispositivo y de cómo esté configurado el navegador (por ejemplo, en una computadora de escritorio, un teléfono móvil, o una tableta).

<Porque es importante>

El viewport es crucial en el diseño web, especialmente cuando se trata de diseño web responsivo,, por lo que el viewport ayuda a los desarrolladores a crear páginas web que se adapten correctamente a cualquier dispositivo.

<Como se usa>

Se utiliza la meta etiqueta viewport en el HTML para especificar cómo debe comportarse el viewport

<Ejemplos>

width=device-width: Establece el ancho del viewport igual al ancho de la pantalla del dispositivo. Esto asegura que el diseño se adapte al tamaño de la pantalla del dispositivo.

<Ejemplo #1>

initial-scale=1.0: Define el nivel de zoom inicial cuando la página se carga por primera vez. Un valor de 1.0 significa que la página no está ampliada ni reducida.

<Que ocurre si no se usa>

El viewport es el área visible de la página web en una ventana del navegador. Es fundamental para el diseño web responsivo. La meta etiqueta viewport controla cómo se presenta la página en diferentes dispositivos, asegurando una mejor experiencia de usuario.
